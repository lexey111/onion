# Onion

Small application example to demonstrate Layered architecture

* API
* Data Access / Query Cache
* Services
* Global State
* Local States
* Local sub-states
* UI

Includes
* Vite + React + Typescript
* TanStack Query (React Query)
* Supabase
* MUI
* MobX
